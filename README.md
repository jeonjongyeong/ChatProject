# 팀프로젝트 체팅 사이트 
- 회원 가입이 가능하다.
- 로그인 가능하다.
- 게시판의 게시물 작성, 수정, 삭제는 로그인 후 이용가늘하다.
- 조회시 조회수가 증가한다.
- 수정, 삭제는 작성자만이 가능하다.
- 비밀번호는 암호화되어 저장된다.

# 체팅서버는 따로 만들어져있음
- 체팅의 작성 이용 가능하다.
- 가명으로 전체 쳇팅의 참가 가능하다.
